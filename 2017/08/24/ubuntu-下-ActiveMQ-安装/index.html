- 创建启动脚本

vagrant@ubuntu:~$ sudo service activemq start
INFO: Loading ‘/etc/default/activemq’
INFO: Using java ‘/usr/bin/java’
INFO: Starting - inspect logfiles specified in logging.properties and log4j.properties to get details
INFO: changing to user ‘activemq’ to invoke java

vagrant@ubuntu:~$ sudo netstat -naptu | grep 61616
tcp6       0      0 :::61616                :::*                    LISTEN      3540/java


 vagrant@ubuntu:~$ sudo netstat -naptu | grep 8161

tcp6 0 0 :::8161 :::* LISTEN 3540/java
```

- 可以使用 `http://ip:8161` 访问 ActiveMQ 的控制台，其中 ip 为 ActiveMQ 所在的 ip 地址。
