getTime-vs-now

发表于 2017-07-06

获取当前的时间

Date.now() 和 new Date().getTime() 都可以用来获取当前的时间，它们返回的结果是相同的，但是 Date.now() 的速度与 getTime() 相比速度更快一些。

使用 Date 对象创建的实例在其创建的那一刻起，它的值就已经冻结了。
